# Google News Clone - Frontend

This is the React frontend for the Google News Clone. It features a clean, responsive interface inspired by Google News, built with TypeScript and Tailwind CSS.

## üöÄ Tech Stack
* **React** (v18+)
* **TypeScript**

* **Tailwind CSS** (Styling)

## ‚ú® Features
* **Responsive Grid Layout:** Displays news in a clean 3-column grid.
* **Live Search:** Search for news topics instantly.
* **Category Filtering:** Filter news by categories (Tech, Sports, Business, etc.).
* **Load More:** "Infinite scroll" style pagination button.
* **Error Handling:** Automatically filters out broken articles or "Client Challenge" errors.
* **Fallback Images:** Displays placeholder images if the article image fails to load.
